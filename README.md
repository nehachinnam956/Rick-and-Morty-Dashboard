# 👽 Rick and Morty Dashboard

A responsive web application that displays characters from the **Rick and Morty API** using JavaScript's Fetch API. The application presents character information in visually appealing cards with images, status, species, gender, origin, and current location.

---

## 🚀 Features

- Fetches live character data from the Rick and Morty API
- Displays character cards dynamically
- Responsive card layout using CSS Grid
- Character image
- Character name
- Status (Alive, Dead, Unknown)
- Species
- Gender
- Origin
- Current Location
- Hover animations
- Error handling for failed API requests

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Rick and Morty API
